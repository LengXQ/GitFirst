Git的简单命令操作

(1) git --version //查看当前git安装成功没有,安装成功显示版本号

(2) git init //初始化当前文件夹

(3) git add . //添加所有的文件,把所有的文件都添加到队列当中去

(4) git add *.html //添加某一类文件

(5) git commit -m"描述提交的代码" //确认提交

(6) git status //查看当前状态

(7) git log //查看提交日志

(8) git reflog //查看所有的日志,包括回滚的日志

(9) git config user.email //查看设置的邮箱

(10) git config user.name //查看设置的用户名

(11) git config --global --unset user.email git config --global --unset user.name //退出登录

(12) git reset --hard 版本号 //版本回滚

(13) git log --oneline //查看一行类型的日志,包括版本号

(14) git clone //仓库地址

(15) add commit //提交到本地仓库

(16) git push //推送到远程仓库

(17) git branch //查看现有分支

(18) git branch 分支名 //创建分支

(19) git checkout 分支名 //切换分支

(20) git merge 分支名 //将指定分支合并到当前分支分支

(21) git branch -d //删除分支
